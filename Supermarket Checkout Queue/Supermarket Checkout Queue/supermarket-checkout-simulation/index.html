System Description

Objective

Model and evaluate the performance of a supermarket checkout system to identify bottlenecks and optimize customer wait times.

Model Type

M/M/3 Queue - Markovian arrivals, Markovian service, 3 parallel servers with a shared queue (FCFS discipline).

Parameters

  • Arrival Rate (λ): 1 customer/min
  • Service Rate (μ): 0.4 customer/min per server
  • Traffic Intensity (ρ): 0.83

Key Findings

  • System operates at healthy 79% utilization
  • Wait times can exceed service times during peaks
  • Workload is balanced across counters
